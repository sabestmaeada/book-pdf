# 📋 Brief Form — บอก AI ช่วยสร้างเทมเพลตหนังสือ

> สำหรับผู้เขียน/ผู้จัดทำหนังสือ ที่ **ไม่ต้องรู้ CSS** ก็ทำได้
>
> วิธีใช้: เปิดไฟล์นี้ → ติ๊กตัวเลือก → copy ทั้งหมด → paste ใน ChatGPT / Claude / Gemini → ส่ง

---

## ⚠ สำคัญก่อนเริ่ม — ต้องแนบไฟล์อะไรกับฟอร์มนี้

**AI ในเว็บไม่เห็นไฟล์ในเครื่องคุณ** → ต้อง paste content ไฟล์ตัวอย่าง 1 ไฟล์ ให้ AI ดูโครงสร้าง

เลือกไฟล์ตัวอย่างที่ใกล้กับสิ่งที่คุณต้องการ (จะแก้สีและฟอนต์ทีหลังก็ได้):

| ไฟล์ | มู้ด | เลือกใช้เมื่อ |
|---|---|---|
| `template-print.gemini.css` | โมเดิร์น-มินิมัล | สไตล์เทคโนโลยี ธุรกิจ |
| `EXAMPLES/example_classic.css` | คลาสสิก-ทางการ | วรรณกรรม ตำราสำคัญ |
| `EXAMPLES/example_workbook.css` | วิชาการ-textbook | ตำราเรียน คู่มือ เตรียมสอบ |
| `EXAMPLES/example_minimal.css` | มินิมัล B&W | นิตยสาร self-help |

### วิธีแนบ — เลือก 1 ทาง

**A. Upload ไฟล์ (แนะนำ)** — ChatGPT-4o / Claude Pro / Gemini รองรับ
1. ลาก-วาง ไฟล์ `BRIEF-FORM.md` (อันนี้) + 1 ไฟล์ `.css` ตัวอย่าง
2. พิมพ์ "ทำตาม BRIEF-FORM โดยใช้โครงสร้างจาก CSS ที่แนบ"

**B. Paste content (ทำงานกับทุก AI)**
1. เปิดไฟล์ `.css` ตัวอย่าง → copy ทั้งไฟล์ → paste บน chat
2. ใต้นั้น paste ฟอร์มนี้ที่ตอบแล้ว
3. ส่ง

## บริบท

**โครงสร้างต้นแบบอยู่ใน CSS ที่ผมแนบ/paste มาให้ (ด้านบน)** — มี 17 sections:
1. Fonts (@import)
2. Variables (`--tpl-*`)
3. Body (html, body, .content p, ul, ol, li)
4. Headings (h1-h4)
5. Heading→p spacing
6. Chapter header (.ch-hdr, .ch-num, .ch-title)
7. Inline elements (strong, em, mark, kbd)
8. OL counter (.content ol li::before)
9. Quiz reset (.bd-grid.quiz)
10. Drop cap (.drop-cap::first-letter)
11. Code blocks (.code-block, .line, .inline-code)
12. Note (.note, .note-label)
13. Blockquote
14. Tables (th, td)
15. TOC (.toc-*)
16. Preface (.preface-*)
17. Image markers + figcaption + hr

## กฎสำคัญ

- ใช้ prefix `--tpl-*` สำหรับ CSS variables
- ทุก rule ต้องมี `!important`
- **ห้ามแตะ**:
  - `@page` (geometry อยู่ใน sizes/)
  - `.cover-image`, `.back-cover-image` (geometry อยู่ใน sizes/)
  - `.credits-page` min-height (อยู่ใน sizes/)
  - `.bd-row` / `.bd-col` `display: table` (ระบบ grid)
  - `.ch-hdr` `display: flex` (chapter layout — ถ้าอยากเปลี่ยน ให้ตั้ง `display: block`)
  - `counter-reset` / `counter-increment` ของ `.content ol` (Thai counter mode)
  - `@counter-style thai-alpha` (อยู่ใน base)
  - `string-set: chapter-title` (running header)
- **WeasyPrint ไม่รองรับ** (จะถูก ignore + warning):
  - `box-shadow`, `text-shadow`, `print-color-adjust`
  - `width: fit-content` (ใช้ block + `::after` underline แทน)
  - gradient text (`background-clip: text`)
  - transform ส่วนใหญ่
- ฟอนต์ทุกตัวต้องมี Thai fallback (`'Sarabun', 'IBM Plex Sans Thai'`)
